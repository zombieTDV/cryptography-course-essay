#

## Installation Guide

### Prerequisites

- Python **3.9 or higher** (3.12 recommended)
- Git installed

---

### Step 1: Create Virtual Environment

```bash
python -m venv .venv
```

Activate the virtual environment:

- On Windows (PowerShell):

```bash
.venv\Scripts\Activate.ps1
```

- On macOS/Linux:

```bash
source .venv/bin/activate
```
